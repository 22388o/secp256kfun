# ECDSAFun!

Generate and verify ECDSA signatures on secp256k1.
Uses [secp256kfun].

## Use

``` toml
[dependencies]
ecdsa_fun = "0.1"
```

### Should use?

This library and [secp256kfun] is experimental.
Not well reviewed or tested.

### Extra Features

- ECDSA Adaptor signatures

[secp256kfun]: https://docs.rs/secp256kfun
